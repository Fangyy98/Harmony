UI Semantic Group Detection: Grouping UI Elements with Similar Semantics in Mobile Graphical User Interface

编号：H043
分类：应用测试与质量
相关性：移动应用 UI 质量相关

为什么收录：
该工作关注移动 GUI 中文本、控件和图片并非孤立工作，而是按交互功能和视觉信息形成语义组件组。论文提出 UISCGD，在 deformable-DETR 基础上加入 UI 元素颜色表示和组分布先验，用 1988 个 iOS/Android 移动 GUI 截图训练语义组检测器。它比单个控件检测更接近布局结构理解，可用于 UI-to-code、可访问性数据生成、感知组检索和 ArkUI 页面结构质量分析。
