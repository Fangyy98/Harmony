为什么收录：
直接把 GUI 元素检测视作特定领域目标检测问题，系统比较多种最新 YOLO 模型在 GUI 元素检测任务上的表现。论文指出 GUI 截图相比自然图像有目标密集、重叠近邻多、类别少但实例多等特点，因此通用检测模型结论不能直接迁移。该工作适合作为 OpenHarmony/ArkUI 控件检测模型选型和 YOLO 系列基线评估的入口。
