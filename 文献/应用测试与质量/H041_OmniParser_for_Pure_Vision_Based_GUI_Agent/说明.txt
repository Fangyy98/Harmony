为什么收录：
OmniParser 将 UI 截图解析为结构化元素，重点解决可交互图标检测、元素语义理解和动作区域绑定问题。它构建交互图标检测数据和图标描述数据，训练检测模型与 caption 模型，并在 ScreenSpot、Mind2Web、AITW 等任务上提升纯截图输入的 GUI agent 能力。对没有 DOM/控件树时的 OpenHarmony 截图级 UI 元素检测和布局结构恢复很有参考价值。
